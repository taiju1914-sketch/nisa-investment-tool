LINE通知設定

毎朝10時（平日）に下落率と投資推奨をLINEで受け取れます。一度設定するだけで自動通知されます。

設定手順（約10分）
① LINE Developersコンソール を開いてLINEアカウントでログイン
② 「プロバイダー作成」→ 任意の名前（例：個人）で作成
③ プロバイダー画面で「チャンネルを作成」→「Messaging API」を選択
④ チャンネル名などを入力して作成完了後、チャンネル設定を開く
⑤ 「Messaging API設定」タブ →「チャンネルアクセストークン（長期）」→「発行」してコピー
⑥ 「Basic settings」タブ →「あなたのユーザーID」（U で始まる文字列）をコピー
⑦ 同ページのQRコードをLINEアプリでスキャンしてBotを友だち追加
⑧ GitHubのシークレット設定 で以下を2件追加：
   ・Name: LINE_CHANNEL_TOKEN → ⑤でコピーしたアクセストークン
   ・Name: LINE_USER_ID → ⑥でコピーしたユーザーID

設定後、ActionsタブでUpdate Price Dataを手動実行するとすぐにLINEが届きます。

🔄 LINE通知しきい値を自動同期

GitHubトークンを登録すると、ステップ6の設定変更が自動でLINE通知に反映されます。

トークンの取得方法
① GitHub Fine-grained token発行ページ を開く
② Token name：「NISA通知」など好きな名前を入力
③ Expiration（有効期限）：任意（長めに設定推奨）
④ Repository access：「Only select repositories」→「nisa-investment-tool」を選択
⑤ Permissions（権限）を以下の2つに変更：
   ・Contents：Read and write（設定の自動同期に必要）
   ・Actions：Read and write（テスト送信ボタンに必要）
⑥ 「Generate token」→ 表示されたトークン（github_pat_ から始まる文字列）をコピー
⑦ 上の入力欄に貼り付けて「保存」を押す
